## 🛠 Tecnologias

- HTML5 + CSS3 vanilla (sem frameworks)
- React 18 + Babel (painel de Tweaks)
- Fontes: Anton · Kaushan Script · Space Grotesk (Google Fonts)
- Animações: CSS `@keyframes`, `IntersectionObserver`, parallax
